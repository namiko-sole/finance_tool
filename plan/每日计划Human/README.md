## 📊 情绪周期判断

### 周期阶段

| 阶段 | 特征 | 策略 | 仓位 |
|------|------|------|------|
| **启动期** | 新题材出现，赚钱效应初现 | 试错新题材，小仓位参与 | 30-50% |
| **加速期** | 龙头连板，赚钱效应强 | 大胆接力，重仓出击 | 70-100% |
| **分歧期** | 高低位切换，炸板增多 | 控制仓位，快进快出 | 30-50% |
| **退潮期** | 高度下降，亏钱效应 | 空仓观望，等待新周期 | 0-10% |
| **冰点期** | 涨停家数少，情绪冰点 | 等待回暖，试错新题材 | 10-30% |

### 判断指标
- **涨停家数**：>30强势，<10弱势
- **连板高度**：5板+强，3板中，1板弱
- **炸板率**：<30%正常，>50%风险
- **昨日涨停溢价**：>2%强，<0%弱

## 🔥 核心战法

### 1. 打首板
- **条件**：热点题材龙头 + 快速拉升（<5分钟）+ 量比>3 + 封单>5000万
- **操作**：涨停价扫板，炸板立即止损
- **风险**：炸板当天亏损10%+，需要70%+胜率

### 2. 接力板（二板、三板）
- **弱转强**：昨日烂板 → 今日强开（高开>3%）→ 快速封板
- **强更强**：昨日一字板 → 今日继续一字或快速封板
- **回封**：早盘炸板 → 分时企稳 → 再次封板

### 3. 龙头战法
- **识别**：率先涨停 + 高度最高 + 带动板块 + 封板坚决
- **操作**：重仓出击，持有到分歧，分批止盈
- **原则**：龙头不死，信仰不减

## 🛡️ 风控原则

### 仓位管理
| 市场状态 | 总仓位 | 单票上限 |
|---------|--------|----------|
| **强势市** | 80% | 20% |
| **震荡市** | 50% | 15% |
| **弱势市** | 30% | 10% |
| **退潮期** | 0-10% | 不操作 |

### 止损纪律
- **打板炸板**：次日开盘止损
- **低吸失败**：跌破支撑止损
- **连续亏损**：空仓休息

### 止盈策略
- **打板成功**：次日高开止盈
- **连板成功**：分歧期止盈
- **达到目标**：分批止盈

### 风控红线
- 单日最大亏损：-3%
- 单周最大亏损：-10%
- 连续3日亏损：空仓

- **持续复盘总结**：每天记录，总结经验

## 🎓 核心心法

### 交易哲学
1. **顺势而为**：上升趋势做多，不找顶；下降趋势空仓，不抄底
2. **快进快出**：不追求买在最低，卖在最高；落袋为安，不贪婪
3. **控制风险**：单票不超过20%，严格止损，连续亏损空仓
4. **保持理性**：不被情绪左右，不被消息误导，坚持系统

### 成功要素
1. **情绪周期判断**：知道自己在周期中的位置
2. **题材持续性**：区分S级（政策）、A级（事件）、B级（技术）、C级（情绪）
3. **龙头股识别**：率先涨停、高度最高、带动板块、封板坚决
4. **风险控制**：仓位控制、止损纪律、止盈策略
