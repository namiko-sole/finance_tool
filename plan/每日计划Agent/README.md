## 🤖 Agent任务说明

### Agent的职责
✅ **DO**：
- 读取时段文件了解任务
- 获取指定的市场数据
- 进行数据分析
- 按指定格式推送结果

❌ **DON'T**：
- 不执行交易操作
- 不给出具体买卖建议（仅提供数据和分析）
- 不代替人类决策

### 时段文件结构

每个时段文件包含4个部分：

1. **📊 数据获取**
   - 必须获取哪些数据
   - 数据来源（财经工具、系统）

2. **🔍 分析要求**
   - 需要进行什么分析
   - 分析维度和标准

3. **📤 推送要求**
   - 推送格式模板
   - 推送时机

4. **⚙️ 可用工具**
   - 可以使用的工具
   - 系统功能

### 推送格式规范

所有推送应遵循以下格式：

```markdown
🔔 **时段标题**（时间）

**核心数据**：
- 数据1：值
- 数据2：值

**分析结果**：
[分析内容]

**操作建议**：
[建议内容]
```

## 🎯 交易风格定位

### 核心特征
- **风格**：游资短线/超短线
- **持股时间**：1-3天
- **决策依据**：情绪周期 + 技术分析 + 数据验证
- **操作节奏**：快进快出，不恋战

### 情绪周期判断

| 阶段 | 特征 | 策略 | 仓位 |
|------|------|------|------|
| **启动期** | 新题材出现 | 试错新题材 | 30-50% |
| **加速期** | 龙头连板 | 大胆接力 | 70-100% |
| **分歧期** | 高低位切换 | 控制仓位 | 30-50% |
| **退潮期** | 高度下降 | 空仓观望 | 0-10% |
| **冰点期** | 涨停家数少 | 等待回暖 | 10-30% |

### 判断指标
- **涨停家数**：>30强势，<10弱势
- **连板高度**：5板+强，3板中，1板弱
- **炸板率**：<30%正常，>50%风险
- **昨日涨停溢价**：>2%强，<0%弱

## 📋 使用流程

### 对于Agent（子任务）
1. Cron触发 → 读取对应时段文件
2. 根据"数据获取"部分获取数据
3. 根据"分析要求"部分进行分析
4. 根据"推送要求"部分推送结果
5. 任务完成

### 对于小何（人类交易员）
1. **盘前**：查看财经快讯汇总、集合竞价观察
2. **盘中**：查看开盘冲刺、板块强化、个股机会等推送
3. **盘后**：查看深度复盘、明日计划
4. **决策**：根据Agent推送的数据和分析，做出交易决策

## 💡 重要提醒

**Agent不是交易员，是分析师**

- Agent提供数据和分析
- 人类交易员做决策
- 保持人机协作模式

**市场永远在，机会永远有**

- 保住本金，才能持续盈利
- 稳定盈利，比暴赚更重要
- 心态稳定，比技术更重要
